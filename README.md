# dlcontract

### Docs

Download verified smart contracts from Etherscan

```
npx dlcontract <ETHERSCAN_URL> [output/]

# example
npx dlcontract https://etherscan.io/address/0x6b175474e89094c44da98b954eedeac495271d0f#code
```
